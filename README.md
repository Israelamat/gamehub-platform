## 🎯 The Vision

**GameHub** is an all-in-one ecosystem designed for gamers and developers alike. It’s not just an e-commerce platform; it's a bridge between playing world-class titles and learning how to build them. 

By integrating **Steam's massive library** with a **community-driven marketplace** and professional **game development courses**, GameHub empowers users to transition from players to creators.

* **Steam Sync**: Browse and filter the biggest titles directly from the Steam API.
* **Community Forge**: A dedicated space for indie creators to showcase and sell their own games.
* **Dev Academy**: Curated video courses focused on Unity, Unreal Engine, and C# development.
* **Smart Discovery**: A recommendation engine that suggests games based on your playstyle and dev interests.

## 🛠️ Tech Stack & Key Architectures

Built with cutting-edge Angular technologies to ensure a smooth, high-performance "gaming" feel.

| Feature | Technology | Description |
| :--- | :--- | :--- |
| **State Management** | `Angular Signals` | High-performance reactive state for filters and search |
| **Dynamic UI** | `LinkedSignal` | Synchronizing local component state with global game services |
| **Visual Effects** | `ScrollReveal` | Cinematic entry animations for game cards and elements |
| **Styling** | `CSS + Glassmorphism` | Modern "Dark Tech" UI with blur effects and neon accents |
| **Rendering** | `OnPush Strategy` | Optimized change detection for 60fps-like navigation |
